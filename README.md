 # JTGeats — Authentic Home Food
**Responsive static website template for home-cooked food delivery**
---
##  About
JTGeats is a clean, mobile-friendly landing website template designed for showcasing home-cooked meals, connecting customers with sellers, and enabling dish requests. Built with plain HTML, CSS, and a bit of JS, it’s lightweight and easy to extend.
---
##  Features
- Hero section with search bar and illustration  
- Menu grid showcasing home kitchen items with ratings and prices  
- Carousel of popular items  
- Video / About section with play overlay  
- Contact form + “Request a Dish” modal popup  
- Responsive layout for desktop, tablet, and mobile  
- Simple “Add to Cart” button behavior with count display
